Team Composer

Descrição do Projeto

O app Team Composer, foi idealizado atráves de uma das matérias inclusas na antiga grade do curso de Software da UTFPR-DV, matéria esta de nome "Fábrica de Software", qual objetivo é realizar o sorteio balanceado de uma equipe com todos os componentes necessários para a entrega de um MVP. O desenvolvimento desse projeto foi dividido entre os alunos pelos professores do semestre, eu específicamente cuidei de toda a parte de UX/UI, qual foram feitos wireframes utilizando a ferramenta Balsamic e os protótipos de alta, utilizando o Figma. O app foi dividido em dois fluxos, o que os alunos teriam acesso e o que os professores utilizariam para sortear os grupos.

Tecnologias Utilizadas

  • BALSAMIC
  • FIGMA
